# Huong dan chay du an

## Backend (Spring Boot)
1. Cai dat Java 17 va MySQL.
2. Tao DB ten `motel` (hoac de Spring tu tao).
3. Sua thong tin DB trong `backend/src/main/resources/application.properties`:
   - `spring.datasource.username`
   - `spring.datasource.password`
4. Chay backend:
   - `cd backend`
   - `mvn spring-boot:run`

Tai khoan mac dinh:
- Username: `admin`
- Password: `admin123`

## Frontend (Next.js)
1. Cai dat Node.js 18+.
2. Cai thu vien:
   - `cd frontend`
   - `npm install`
3. Chay:
   - `npm run dev`

Frontend mac dinh goi API tu `http://localhost:8080/api`.
